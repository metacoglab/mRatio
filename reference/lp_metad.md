# Calculate the log probability simplex of the metad' model

Calculate the log probability simplex of the metad' model

## Usage

``` r
lp_metad(i, prep)
```

## Arguments

- i:

  an observation index

- prep:

  an object containing the data and model draws

## Value

A vector of joint type 1/type 2 response probabilties for observation
`i` in `prep`
